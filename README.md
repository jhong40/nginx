# nginx
```
nginx -v  # print ver
nginx -V  # print ver, compiler ver and conf parm
nginx -t  # don't run , just test config file
nginx -T  # same as -t, dump config files 
nginx -c mnginx.conf  # use mnginx.conf instead of the default nginx.conf
```
